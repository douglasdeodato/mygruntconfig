# mygruntconfig

with this gruntfile you will be able to run all theses tasks bellow.

# grunt
```
 "devDependencies": {
  "grunt": "*",

    "grunt-contrib-htmlmin": "*",
    "grunt-contrib-jade": "*",
    "grunt-contrib-jasmine": "*",
    "grunt-contrib-jshint": "*",
    "grunt-contrib-sass": "*",

```

Install grunt and npm.
http://gruntjs.com/installing-grunt
$ npm install //this will install all the packages needed  from the json file

# How to use:

open your terminal and run the following commands lines.

```
$ grunt (default)
$ grunt watch
$ grunt concat
$ grunt handlebars
$ grunt uglify
$ grunt jasmine
$ grunt jshint
$ grunt htmlmin
$ grunt imagemin
$ grunt sprite
$ grunt sass
$ grunt sass:dev
$ grunt sass:dist
$ grunt jade
$ grunt serve

```
